# @uxwb React UI Components [![NPM](https://img.shields.io/npm/v/@uxwb/react-components.svg)](https://www.npmjs.com/package/@uxwb/react-components)

## Install

```bash
yarn add @uxwb/react-components
```

## Usage

```tsx
import React, { Component } from 'react';

import { Row, Col } from '@uxwb/react-components';

class Example extends Component {
    render() {
        return (
            <Row>
                <Col count={2}>...</Col>
            </Row>
        );
    }
}
```
