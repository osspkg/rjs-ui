# @uxwb React UI Components [![NPM](https://img.shields.io/npm/v/@uxwb/rjs-ui.svg)](https://www.npmjs.com/package/@uxwb/rjs-ui)

## Install

```bash
yarn add @uxwb/rjs-ui
```

## Usage

```tsx
import React, { Component } from 'react';

import { Row, Col } from '@uxwb/rjs-ui';

class Example extends Component {
  render() {
    return (
      <Row>
        <Col count={2}>...</Col>
      </Row>
    );
  }
}
```
